# Dashboard Images

This directory contains the visual assets for the dashboard hero section.

## Image Files

### hero-bg.png
- **Purpose**: Background image for the hero section
- **Current Status**: Placeholder image (solid color)
- **Recommended**: Replace with actual virus-themed background image
- **Specifications**: 
  - Recommended size: 1200x400px or larger
  - Format: PNG or JPG
  - Should work well with gradient overlay

### feevale-logo.png
- **Purpose**: Universidade Feevale logo displayed in the hero section
- **Current Status**: Placeholder image (solid white)
- **Recommended**: Replace with official Feevale logo
- **Specifications**:
  - Recommended size: 300x100px or similar aspect ratio
  - Format: PNG (with transparency preferred)
  - Should be readable against the hero background

## How to Replace Images

Simply replace the placeholder files with your actual images:

```bash
# Replace hero background
cp /path/to/your/virus-background.png dashboard/hero-bg.png

# Replace Feevale logo
cp /path/to/your/feevale-logo.png dashboard/feevale-logo.png
```

The CSS is already configured to display these images properly with responsive sizing and styling.
